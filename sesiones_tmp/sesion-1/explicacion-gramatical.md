# Explicación gramatical – Estilo indirecto en pasado

## 1) Cambios de tiempo verbal
Cuando el verbo introductor está en pasado (dijo, explicó, contó, preguntó), el tiempo del verbo subordinado suele retroceder:

| Estilo directo | Estilo indirecto |
|---|---|
| Presente | Imperfecto |
| Pretérito perfecto | Pluscuamperfecto |
| Pretérito indefinido | Pluscuamperfecto |
| Futuro | Condicional |
| Imperativo | Pretérito imperfecto de subjuntivo / infinitivo |
| Presente de subjuntivo | Imperfecto de subjuntivo |

**Ejemplos**
- “Estoy cansado.” → Dijo que **estaba** cansado.
- “He perdido el móvil.” → Dijo que **había perdido** el móvil.
- “Llegaré mañana.” → Dijo que **llegaría** al día siguiente.
- “Ven pronto.” → Me pidió que **viniera** pronto.

---

## 2) Cambios de deícticos (tiempo/lugar/persona)

| Directo | Indirecto |
|---|---|
| hoy | aquel día |
| ayer | el día anterior |
| mañana | al día siguiente |
| ahora | entonces |
| aquí | allí |
| este/esta | ese/esa / aquel/aquella |
| mi/tu | su |

**Ejemplo**
- “Estoy aquí hoy con mi hermano.” → Dijo que **estaba allí aquel día con su hermano**.

---

## 3) Verbos de habla alternativos
Usa verbos variados, no siempre “decir”:
- **preguntar** (preguntó si / qué / cuándo...)
- **responder / contestar**
- **explicar / comentar / aclarar**
- **pedir / rogar / suplicar**
- **advertir / avisar**
- **prometer**

---

## 4) Errores frecuentes
- Mantener el tiempo verbal original (incorrecto): *Dijo que está cansado.*
- Olvidar cambios deícticos: *Dijo que llegaría mañana.*
- No cambiar pronombres: *Dijo que te ayudaba.*

---

## 5) Mini guía rápida
1. Identifica el verbo introductor en pasado.
2. Retrocede el tiempo verbal.
3. Cambia deícticos (tiempo/lugar/persona).
4. Ajusta pronombres.

---

## 6) Micro-ejercicio de ejemplo
Directo: “He terminado el proyecto y te lo envío hoy.”  \
Indirecto: Dijo que **había terminado** el proyecto y que **me lo enviaría aquel día**.
